Flyt vindue uden for skærmen

Specielt når man sidder med en lille skærm kan man komme ud for at skærmen er for lille til at vise et vindue. Ved at trykke ALT og flytte vinduet med musen.

Hjerter fri og andre kortspil

Mange har spurgt et at kunne spille hjerter-fri. Uden at kende spillet ser det ud til at pakken gnome-hearts indeholder spillet. Installeres følgende kommando:

sudo apt install gnome-hearts

I pakkerne kpat og pysolfc findes et hav af andre kortspil. Installeres følgende kommandoer:

sudo apt install kpat sudo apt install pysolfc

Pakken sgt-puzzles indeholder en masse små logik- og pusle-spil. Installeres følgende kommando:

sudo apt install sgt-puzzles
